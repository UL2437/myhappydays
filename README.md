## 起点

### 背景

算是命中注定来二外了。（摇号+分数够+提前被录取）\
虽然说也去看了看其他学校，并且我是有机会的，但我这个人趋于保守，这个选择已经不错了。

人真的不要美化自己没选择的路，五年后这句话会再一次敲响警钟的。

入学的时候知道二外有信息学竞赛，作为做过那些项目，自学 **C** 语言而且还在学习 **C++** 的人肯定还是感兴趣的，毕竟了解到竞赛也是用 **C++** ，不过没有进一步了解。

开学选社团，我选了机器人的。第一节课发现其实它是魔改的单片机，还是要额外购买器材的，不过很快就跟我无关了。

当天放学我爸问我选的什么社团，我告诉他之后他说我们其实还有竞赛，但是在我的视角看当时我报社团的时候根本没有竞赛这个选项！我还以为社团就是社团。然后我爸问我怎么想，我不以为意，说了一句话：

>是金子总会发光的。

我爸有点生气，就说还是得选择正确的路，并且已经把我调到竞赛去了，~~那为什么不直入主题呢？直到今天我回忆还是感觉莫名其妙。~~我想那也好，我可以同时在机构和学校学 **C++** 了。

那么第二次社团课，我便去机房上竞赛了。

### 入门

总共有三十多个同学，其中还有我的同班同学HZY（尊重别人，这里就用缩写了）和小学同学LSR，上课的老师姓何。

他带我们去小司码（二外的合作OJ）上注册账号。

然后我们当时敲代码的时候流行什么呢？遇到代码挂分(80~90)我们就会打开数据写一个if特判，我当时个人签名就是这个：

>当你的代码是90分时怎么办？用万能的if语句！

确实是不求甚解了，哈哈。

期间当时何老师并没有教我们用 $(x+y-1)/y$ 的向上取整方式（批判×1）所以当时LZX展示的解决方案是

```cpp
=x/y+0.999999999999999
```

这就是我在九年级年级以前的写法了，虽然说机构的老师教过 $(x+y-1)/y$ 的写法但当时没怎么看懂（？？？）看样子入门的教练还是很重要的（批判×2）

#### LSR名场面 I

然后LSR某一次在写题的时候用这种方法向上取整，但是他发现错了于是来问我，我看了他的代码感觉好像没问题，于是有点懵，想了一会之后我说要不要多加几个9试试？他改了之后拍手说“对了！”，提交的时候他又加了几个9并且对其说明是“以防万一”， **AC** 之后我们两个就在那笑。
